# Vishnuvardhan Portfolio

A modern, 3D-inspired UI/UX portfolio website showcasing design and development skills.

## Features

- **3D Glassmorphism Design**: Modern card-based layout with glass effect and depth shadows
- **Interactive Hero Section**: Animated gradient orbs and floating cards
- **Responsive Design**: Fully responsive and mobile-optimized
- **Smooth Animations**: CSS animations and transitions for enhanced user experience
- **Modern Styling**: Uses CSS custom properties and CSS Grid/Flexbox
- **Dark Theme**: Sleek dark mode with vibrant accent colors

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Advanced features including:
  - CSS Grid & Flexbox
  - CSS Custom Properties (Variables)
  - CSS Animations & Transitions
  - Backdrop Filters & Gradients
  - Transform 3D effects
- **JavaScript**: Optional for enhanced interactivity

## Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/VishnuvardhanRoy/vishnuvardhan-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vishnuvardhan-portfolio
   ```

3. Open `index.html` in your web browser

## Customization

### Edit Colors
Open `style.css` and modify the CSS variables in the `:root` selector:

```css
:root {
  --bg: #020617;              /* Background color */
  --accent: #38bdf8;          /* Primary accent */
  --text: #e5e7eb;            /* Text color */
  --muted: #9ca3af;           /* Muted text */
  /* ... more variables */
}
```

### Update Content
Edit the sections in `index.html` to add your own:
- Personal information
- Project descriptions
- Contact details
- Social links
